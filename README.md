# Pet Sus Landing Page

Site estático premium para a Pet Sus | Clínica Veterinária Popular 24h em Fortaleza.

## Estrutura

- `index.html`: página principal com CSS e JavaScript embutidos.
- `assets/images/`: imagens usadas no site.
- `assets/video/`: vídeo original do plano Petlove.
- `docs/`: prompts e instruções de apoio.

## Atualizações comuns

- Trocar imagem principal: substitua `assets/images/petsus-hero.png`.
- Trocar logo: substitua `assets/images/petsus-logo.jpg`.
- Trocar imagem do plano: substitua `assets/images/petsus-plano.png`.
- Editar telefone, endereço, serviços ou depoimentos: altere os textos diretamente no `index.html`.

## Publicação

O projeto não precisa de build. Pode ser publicado diretamente na Vercel como site estático.
